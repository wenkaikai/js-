字符串的方法

        1, chartAt() 返回指定索引位置字符
        2，charCodeAt() 返回指定索引位置字符的 Unicode 值
        3, concat()  	连接两个或多个字符串，返回连接后的字符串
        4, fromCharCode() 将unicode 转换为字符串
        5，indexOf()  	返回字符串中检索指定字符第一次出现的位置
        6，lastIndexOf() 返回字符串中检索指定字符最后一次出现的位置
        7，localeCompare() 用本地特定的顺序来比较两个字符串
        8，match() 找到一个或多个正则表达式的匹配
        9，replace()  替换与正则表达式匹配的子串
        10,includes(searchValue,start) 设置从那个位置开始查找，默认为 0。@return 如果找到匹配的字符串返回 true，否则返回 false。
        11,repeat()  复制字符串指定次数，并将它们连接在一起返回。es6 @retrun String 
        12，search(searchvalue) 查找与正则表达式相匹配的值。 @return number 与指定查找的字符串或者正则表达式相匹配的 String 对象起始位置
        13, slice() 提取字符串的片断，并在新的字符串中返回被提取的部分。 
        14, split() 把字符串分割为字符串数组。
        15, startsWith()  查看字符串是否以指定的子字符串开头。
        16, substr() 从起始索引号提取字符串中指定数目的字符。
        17, substring() 提取字符串中两个指定的索引号之间的字符。
        18, toLowerCase() toUpperCase() 
        19, trim()  去除字符串两边的空白
        20, toLocaleLowerCase()  toLocaleUpperCase() 根据本地主机的语言环境把字符串转换为小写。
        21, valueOf() 返回某个字符串对象的原始值。
